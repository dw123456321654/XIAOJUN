# 2048游戏 - 项目说明

## 📁 项目目录结构

```
2048游戏/
├── index.html              # 游戏主文件（双击即可运行）
├── README.md               # 项目说明（本文件）
├── cleanup.py              # 临时文件清理脚本
└── docs/                   # 项目文档
    ├── prd/                # 产品需求文档
    │   └── 需求文档.md
    ├── design/             # 设计文档
    │   ├── 技术选型报告.md
    │   ├── 架构设计文档.md
    │   └── UI设计文档.md
    └── project/            # 项目管理文档
        ├── 文件管理规范.md
        ├── 项目计划.md
        └── 风险评估报告.md
```

---

## 📋 文件归属

### 根目录
- `index.html` - 游戏主文件（必需，不删除）
- `README.md` - 项目说明（必需，不删除）
- `cleanup.py` - 临时文件清理脚本（可选，用完可删除）

### docs/ 目录
所有项目文档，按类型分文件夹。

---

## 🎮 如何运行

1. 双击 `index.html` 文件
2. 游戏在浏览器中打开
3. 使用方向键或触摸滑动移动数字

---
