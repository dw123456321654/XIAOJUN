## 技术实现

### 关键算法

#### 1. 数组合并算法

```javascript
function mergeArray(arr) {
  let result = [];
  let skipNext = false;
  let score = 0;

  for (let i = 0; i < arr.length; i++) {
    if (skipNext) {
      skipNext = false;
      continue;
    }

    if (i + 1 < arr.length && arr[i] !== 0 && arr[i] === arr[i + 1]) {
      const merged = arr[i] * 2;
      result.push(merged);
      score += merged;
      skipNext = true;

      if (merged === 2048) {
        setWon(true);
      }
    } else {
      result.push(arr[i]);
    }
  }

  // 补齐0
  while (result.length < arr.length) {
    result.push(0);
  }

  return { result, score };
}
```

**时间复杂度：** O(n)  
**空间复杂度：** O(n)

#### 2. 移动算法（以向左移动为例）

```javascript
function moveLeft() {
  const grid = getGrid();
  let newGrid = [];
  let moved = false;
  let totalScore = 0;

  // 按行处理（0,1,2,3）是一行
  for (let row = 0; row < GRID_SIZE; row++) {
    const rowStart = row * GRID_SIZE;
    const rowEnd = rowStart + GRID_SIZE;
    const currentRow = grid.slice(rowStart, rowEnd);

    const { result, moved: rowMoved, score } = moveRow(currentRow);
    newGrid = newGrid.concat(result);

    if (rowMoved) moved = true;
    totalScore += score;
  }

  return { grid: newGrid, moved, score: totalScore };
}
```

**时间复杂度：** O(n²)  
**空间复杂度：** O(n²)

#### 3. 游戏结束判断

```javascript
function checkGameOver(grid) {
  // 1. 检查是否有空白位置
  if (grid.includes(0)) {
    return false;
  }

  // 2. 检查是否有可合并的相邻数字
  // 水平方向
  for (let row = 0; row < GRID_SIZE; row++) {
    for (let col = 0; col < GRID_SIZE - 1; col++) {
      const i = row * GRID_SIZE + col;
      if (grid[i] === grid[i + 1]) {
        return false;
      }
    }
  }

  // 垂直方向
  for (let row = 0; row < GRID_SIZE - 1; row++) {
    for (let col = 0; col < GRID_SIZE; col++) {
      const i = row * GRID_SIZE + col;
      if (grid[i] === grid[i + GRID_SIZE]) {
        return false;
      }
    }
  }

  return true;
}
```

**时间复杂度：** O(n²)  
**空间复杂度：** O(1)

## 遇到的问题及解决方案

### 问题1：移动算法设计

**问题：** 如何处理多个相同数字的合并？

**解决：** 使用skipNext标志，确保每个数字每轮最多合并一次。例如，[2, 2, 2, 2] 合并后为 [4, 4, 0, 0]，而不是 [8, 0, 0, 0]。

### 问题2：触摸事件误触发滚动

**问题：** 移动端触摸滑动时，会触发浏览器的默认滚动行为。

**解决：** 在touchend事件中调用 `e.preventDefault()`，阻止默认滚动行为。

### 问题3：动画性能优化

**问题：** 最初的动画实现使用了left/top属性，导致性能较差。

**解决：** 改用transform属性，利用GPU加速，动画流畅度大幅提升。
