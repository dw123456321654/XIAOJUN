## 🐛 问题描述

测试人员发现严重的逻辑错误：`mergeArray` 函数补齐0的长度错误，导致：
1. 有空格的数组长度不正确
2. 全空格数组变成空数组 `[]` 而不是 `[0, 0, 0, 0]`
3. 游戏无法正常运行

## 🔧 修复内容

### 修复1：mergeArray 函数

#### 修复前（❌ 错误）
```javascript
function mergeArray(arr) {
  // ... 合并逻辑 ...

  // ❌ 错误：补齐到过滤后数组的长度
  while (result.length < arr.length) {
    result.push(0);
  }

  return { result, score };
}
```

**问题：** `arr.length` 是过滤后的长度，导致补齐不正确。

#### 修复后（✅ 正确）
```javascript
function mergeArray(arr, targetLength = null) {
  let result = [];
  let skipNext = false;
  let score = 0;

  for (let i = 0; i < arr.length; i++) {
    if (skipNext) {
      skipNext = false;
      continue;
    }

    if (i + 1 < arr.length && arr[i] !== 0 && arr[i] === arr[i + 1]) {
      // 合并
      const merged = arr[i] * 2;
      result.push(merged);
      score += merged;
      skipNext = true;

      if (merged === 2048) {
        setWon(true);
      }
    } else {
      // 不合并
      result.push(arr[i]);
    }
  }

  // ✅ 修复：补齐到目标长度
  const lengthToFill = targetLength !== null ? targetLength : arr.length;
  while (result.length < lengthToFill) {
    result.push(0);
  }

  return { result, score };
}
```

**关键修改：**
- 添加 `targetLength` 参数
- 补齐时使用 `targetLength` 而不是 `arr.length`

### 修复2：moveRow 函数

#### 修复前（❌ 错误）
```javascript
function moveRow(row) {
  // 1. 移除0
  let filtered = row.filter(val => val !== 0);

  // 2. 合并相同数字
  const { result: merged, score } = mergeArray(filtered);

  // 3. 判断是否移动
  const moved = JSON.stringify(row) !== JSON.stringify(merged);

  return { result: merged, moved, score };
}
```

**问题：** 没有传递原始长度，导致 `mergeArray` 无法正确补齐0。

#### 修复后（✅ 正确）
```javascript
function moveRow(row) {
  const originalLength = row.length;  // ✅ 保存原始长度

  // 1. 移除0
  let filtered = row.filter(val => val !== 0);

  // 2. 合并相同数字（传递原始长度）
  const { result: merged, score } = mergeArray(filtered, originalLength);

  // 3. 判断是否移动
  const moved = JSON.stringify(row) !== JSON.stringify(merged);

  return { result: merged, moved, score };
}
```

**关键修改：**
- 保存原始长度 `originalLength`
- 调用 `mergeArray` 时传递 `originalLength`
