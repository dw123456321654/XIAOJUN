## 🔴 严重问题：mergeArray 函数语法错误

### 问题位置
第 550-575 行附近

### 错误代码
```javascript
function mergeArray(arr) {
  let result = [];
  let skipNext = false;
  let score = 0;

  for (let i = 0; i < arr.length; i++) {
    if (skipNext) {
      skipNext = false;
      continue;
    }

    if (i + 1 < arr.length && arr[i] !== 0 && arr[i] === arr[i + 1]) {
      // 合并
      const merged = arr[i] * 2;
      result.push(merged);
      score += merged;
      skipNext = true;

      // 检查是否达到2048
      if (merged === 2048) {
        setWon(true);
      }
    } else {
      // 不合并
      result.push(arr[i]);
    }
  }  // ❌ 缺少大括号 }

  // 补齐0
  while (result.length < arr.length) {
    result.push(0);
  }

  return { result, score };
}
```

### 问题说明
**for 循环缺少右大括号 `}`**，导致：
1. JavaScript 语法错误
2. 函数无法正常执行
3. 整个移动逻辑完全失效
