## ❌ 严重问题：补齐0的逻辑错误

### 问题描述
在 `mergeArray` 函数中，补齐0的逻辑有问题，导致数组长度不正确。

### 错误代码
```javascript
function mergeArray(arr) {
  // ... 合并逻辑 ...

  // ❌ 错误：补齐到过滤后数组的长度
  while (result.length < arr.length) {
    result.push(0);
  }

  return { result, score };
}
```

### 问题分析
- `arr.length` 是**过滤后**的数组长度
- 但是 `arr` 本身就是过滤后的数组！
- 所以补齐逻辑实际上**没有起作用**

### 测试结果

#### 测试用例1：基础合并 ✅
- 输入：[2, 2, 4, 4]
- 输出：[4, 8, 0, 0]
- 预期：[4, 8, 0, 0]
- 结果：✅ 正确

#### 测试用例2：有空格 ❌ **严重错误**
- 输入：[2, 0, 2, 4]
- 输出：[4, 4, 0] ❌ 长度错误
- 预期：[4, 4, 0, 0]
- 问题：长度应该是4，但实际是3
- 问题：分数应该是8，但实际是4

**分析：**
```
输入：[2, 0, 2, 4]
↓ filter (移除0)
filtered: [2, 2, 4]
↓ mergeArray
result: [4, 4]
↓ 补齐0 (result.length=2, arr.length=3)
result: [4, 4, 0]  ← ❌ 长度错误！
```

**根本原因：**
- `arr.length` 是 3（过滤后）
- 但是原数组的长度是 4
- 补齐应该到4，而不是3

#### 测试用例3：全空格 ❌ **严重错误**
- 输入：[0, 0, 0, 0]
- 输出：[] ❌ 长度错误
- 预期：[0, 0, 0, 0]
- 问题：长度应该是4，但实际是0

**分析：**
```
输入：[0, 0, 0, 0]
↓ filter (移除0)
filtered: []
↓ mergeArray
result: []
↓ 补齐0 (result.length=0, arr.length=0)
result: []  ← ❌ 没有补齐！
```

#### 测试用例4：无合并 ✅

- 预期：[2, 4, 8, 16]

## 🐛 其他潜在问题

### 问题1：分数计算不准确
在测试用例2中：
- 输入：[2, 0, 2, 4]
- 实际合并：2+2=4（得分4）
- 预期合并：2+2=4（得分4），4没有合并（得分0），总分8

**分析：**
- 2和2合并成4，得4分
- 4和后面的4（位置2和3）没有合并，因为它们不相邻
- 但是按照2048的规则，移动后应该是 [4, 4, 0, 0]
- 4和4应该再次合并吗？

**检查2048规则：**
- 在标准2048中，一行数字向左移动
- [2, 0, 2, 4] 移动后应该是 [4, 4, 0, 0]
- 但是4和4是相邻的，会再次合并成8吗？
- 答案：不会！每个数字每轮最多合并一次

所以得分4是正确的，测试用例2的预期分数应该是4，不是8。

**但是长度问题仍然存在！**

```javascript
function moveRow(row) {
  const originalLength = row.length;  // 保存原始长度

  // 1. 移除0
  let filtered = row.filter(val => val !== 0);

  // 2. 合并相同数字
  const { result: merged, score } = mergeArray(filtered, originalLength);  // 传递原始长度

  // 3. 判断是否移动

  return { result: merged, moved, score };
}
```

### 修复 mergeArray 函数
```javascript
function mergeArray(arr, targetLength = null) {
  let result = [];
  let skipNext = false;
  let score = 0;

  for (let i = 0; i < arr.length; i++) {
    if (skipNext) {
      skipNext = false;
      continue;
    }

    if (i + 1 < arr.length && arr[i] !== 0 && arr[i] === arr[i + 1]) {
      // 合并
      const merged = arr[i] * 2;
      result.push(merged);
      score += merged;
      skipNext = true;

      // 检查是否达到2048
      if (merged === 2048) {
        setWon(true);
      }
    } else {
      // 不合并
      result.push(arr[i]);
    }
  }

  // ✅ 修复：补齐到目标长度（原始数组长度）
  const lengthToFill = targetLength !== null ? targetLength : arr.length;
  while (result.length < lengthToFill) {
    result.push(0);
  }

  return { result, score };
}
```
