## 1. 设计规范

### 1.1 设计原则

1. **简洁清爽**：类似Notion的极简风格，避免视觉干扰
2. **信息清晰**：公式、示例、错误案例等信息层次分明
3. **易于使用**：导航清晰，操作简单，新手也能快速上手
4. **性能优秀**：界面流畅，响应迅速
5. **一致性**：统一的视觉语言和交互模式

│  公式：                                          │
│  ┌─────────────────────────────────────────┐  │  ← 公式

│  └─────────────────────────────────────────┘  │
│  [复制公式]                                      │
│                                                 │
│  结果：001                                       │
│                                                 │
│  详细说明：                                       │
│  • "张三"：要查找的值                           │
│  • A1:C4：查找范围，首列A列包含姓名             │

**设计说明：**

1. **顶部操作栏**

   - 快捷操作：收藏、练习、对比

2. **内容区块**

   - 【语法】：代码块，等宽字体
   - 【参数说明】：列表形式
   - 【示例】：数据表格 + 公式 + 结果 + 说明
   - 【错误案例】：错误示例 + 修正方案
   - 【使用建议】：列表形式

   - 点击【复制示例数据】→ 复制表格数据到剪贴板
