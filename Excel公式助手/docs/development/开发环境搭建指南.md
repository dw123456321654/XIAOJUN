## 2. 安装步骤

### 2.1 安装Node.js

1. 访问 https://nodejs.org/
2. 下载LTS版本（推荐）
3. 运行安装程序
4. 验证安装：

```bash
node --version
npm --version
```

**预期输出：**
```
node --version
v18.x.x

npm --version
9.x.x
```

---

### 2.2 安装Rust

1. 访问 https://rustup.rs/
2. 下载rustup-init.exe
3. 运行rustup-init.exe
4. 按照提示安装（选择默认选项即可）
5. 验证安装：

```bash
rustc --version
cargo --version
```

**预期输出：**
```
rustc --version
rustc 1.70.0 (or higher)

cargo --version
cargo 1.70.0 (or higher)
```

---

### 2.3 安装Git

1. 访问 https://git-scm.com/
2. 下载Windows版本
3. 运行安装程序
4. 验证安装：

```bash
git --version
```

**预期输出：**
```
git version 2.x.x.windows.x
```

---

### 2.4 安装VS Code

1. 访问 https://code.visualstudio.com/
2. 下载Windows版本
3. 运行安装程序

**推荐插件：**

| 插件名称 | 用途 |
|---------|------|
| Volar | Vue 3语法支持 |
| Rust Analyzer | Rust语言支持 |
| ESLint | JavaScript/TypeScript检查 |
| Prettier | 代码格式化 |
| TailwindCSS IntelliSense | TailwindCSS支持 |

### 3.3 初始化Tauri项目

```bash
# 创建Tauri项目（使用pnpm）
pnpm create tauri-app
```

**按照提示操作：**

1. Project name: `excel-helper`
2. Directory: `.`（当前目录）
3. Choose package manager: `pnpm`
4. Choose UI template: `Vue`
5. Choose UI flavor: `TypeScript`
6. Choose your preferred UI library: `Naive UI`

**预期输出：**
```
✔ Project name is excel-helper
✔ Directory is .
✔ Choose package manager is pnpm
✔ Choose UI template is Vue
✔ Choose UI flavor is TypeScript
✔ Choose your preferred UI library is Naive UI

🎉 Tauri project created!

Next steps:
  cd .
  pnpm install
  pnpm tauri dev
```

### 5.3 vite.config.ts

```typescript
import { defineConfig } from 'vite';
import vue from '@vitejs/plugin-vue';
import { fileURLToPath, URL } from 'node:url';

// https://vitejs.dev/config/
export default defineConfig(async () => {
  const tauri = await import('@tauri-apps/plugin');
  
  return {
    plugins: [vue(), tauri.tauri()],
    resolve: {
      alias: {
        '@': fileURLToPath(new URL('./src', import.meta.url))
      }
    },
    build: {
      target: 'esnext',
      minify: 'esbuild',
      sourcemap: false,
      rollupOptions: {
        output: {
          manualChunks: {
            'vendor': ['vue', 'vue-router', 'pinia'],
            'ui': ['naive-ui']
          }
        }
      }
    },
    server: {
      port: 5173,
      strictPort: true,
      watch: {
        ignored: ['**/src-tauri/**']
      }
    }
  };
});
```

## 8. 常见问题

### 8.1 Rust编译错误

**问题：** `error: linker 'link.exe' not found`

**解决方案：**
1. 安装Microsoft C++ Build Tools
2. 下载地址：https://visualstudio.microsoft.com/downloads/
3. 选择"Desktop development with C++"

---

### 8.2 WebView2错误

**问题：** `WebView2 not found`

**解决方案：**
1. 下载并安装WebView2 Runtime
2. 下载地址：https://developer.microsoft.com/microsoft-edge/webview2/
3. 运行安装程序

---

### 8.3 端口被占用

**问题：** `Error: Port 5173 is already in use`

**解决方案：**

**方案1：关闭占用端口的进程**
```bash
# 查看占用端口的进程
netstat -ano | findstr :5173

# 结束进程
taskkill /PID <进程ID> /F
```

**方案2：修改端口号**
```json
// vite.config.ts
server: {
  port: 5174,  // 修改为其他端口
  strictPort: true
}
```

```json
// tauri.conf.json
"build": {
  "devUrl": "http://localhost:5174",  // 修改为对应端口
  // ...
}
```

---

### 8.4 依赖安装失败

**问题：** `npm install` 或 `pnpm install` 失败

**解决方案：**

**方案1：清理缓存**
```bash
# 清理npm缓存
npm cache clean --force

# 清理node_modules
rm -rf node_modules

# 重新安装
pnpm install
```

**方案2：使用国内镜像**
```bash
# 配置淘宝镜像
pnpm config set registry https://registry.npmmirror.com

# 重新安装
pnpm install
```
