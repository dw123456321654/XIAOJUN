# 开发环境检查清单

---

## 环境检查

### 必需软件

| 软件 | 版本要求 | 状态 | 命令 |
|------|---------|------|------|
| Node.js | 18.0+ | ❌ 未检查 | `node --version` |
| Rust | 1.70+ | ❌ 未检查 | `rustc --version` |
| Git | 最新版 | ❌ 未检查 | `git --version` |

### 可选软件

| 软件 | 状态 | 命令 |
|------|------|------|
| WebView2 Runtime | ❌ 未检查 | `reg query "HKLM\SOFTWARE\WOW6432Node\Microsoft\EdgeUpdate\Clients\{F3017226-FE2A-4295-8BDF-00C3A9A7E4C5}"` |
| VS Code | ❌ 未检查 | `code --version` |

---

## 检查步骤

### 1. 检查Node.js

```powershell
node --version
```

**预期输出：**
```
v18.x.x 或更高版本
```

**如果未安装：**
1. 访问 https://nodejs.org/
2. 下载LTS版本
3. 运行安装程序

---

### 2. 检查Rust

```powershell
rustc --version
```

**预期输出：**
```
rustc 1.70.0 或更高版本
```

**如果未安装：**
1. 访问 https://rustup.rs/
2. 下载rustup-init.exe
3. 运行rustup-init.exe

---

### 3. 检查Git

```powershell
git --version
```

**预期输出：**
```
git version 2.x.x.windows.x
```

**如果未安装：**
1. 访问 https://git-scm.com/
2. 下载Windows版本
3. 运行安装程序

---

### 4. 检查WebView2 Runtime

```powershell
reg query "HKLM\SOFTWARE\WOW6432Node\Microsoft\EdgeUpdate\Clients\{F3017226-FE2A-4295-8BDF-00C3A9A7E4C5}"
```

**预期输出：**
```
如果已安装，会显示版本信息
```

**如果未安装：**
1. 访问 https://developer.microsoft.com/microsoft-edge/webview2/
2. 下载并安装WebView2 Runtime

---

### 5. 检查VS Code

```powershell
code --version
```

**预期输出：**
```
1.85.x 或更高版本
```

**如果未安装：**
1. 访问 https://code.visualstudio.com/
2. 下载Windows版本
3. 运行安装程序

---

## 安装必备VS Code插件

打开VS Code，搜索并安装以下插件：

1. **Volar** - Vue 3语法支持
2. **Rust Analyzer** - Rust语言支持
3. **ESLint** - JavaScript/TypeScript检查
4. **Prettier** - 代码格式化
5. **TailwindCSS IntelliSense** - TailwindCSS支持

## 快速检查脚本

将以下命令复制到PowerShell运行：

```powershell
Write-Host "=== 开发环境检查 ===" -ForegroundColor Cyan

# 检查Node.js
Write-Host "`n[1] 检查Node.js..." -ForegroundColor Yellow
$nodeVersion = node --version 2>$null
if ($nodeVersion) {
    Write-Host "  ✅ $nodeVersion" -ForegroundColor Green
} else {
    Write-Host "  ❌ Node.js 未安装" -ForegroundColor Red
}

# 检查Rust
Write-Host "`n[2] 检查Rust..." -ForegroundColor Yellow
$rustVersion = rustc --version 2>$null
if ($rustVersion) {
    Write-Host "  ✅ $rustVersion" -ForegroundColor Green
} else {
    Write-Host "  ❌ Rust 未安装" -ForegroundColor Red
}

# 检查Git
Write-Host "`n[3] 检查Git..." -ForegroundColor Yellow
$gitVersion = git --version 2>$null
if ($gitVersion) {
    Write-Host "  ✅ $gitVersion" -ForegroundColor Green
} else {
    Write-Host "  ❌ Git 未安装" -ForegroundColor Red
}

# 检查WebView2
Write-Host "`n[4] 检查WebView2 Runtime..." -ForegroundColor Yellow
$webview = reg query "HKLM\SOFTWARE\WOW6432Node\Microsoft\EdgeUpdate\Clients\{F3017226-FE2A-4295-8BDF-00C3A9A7E4C5}" 2>$null
if ($webview) {
    Write-Host "  ✅ WebView2 Runtime 已安装" -ForegroundColor Green
} else {
    Write-Host "  ❌ WebView2 Runtime 未安装" -ForegroundColor Red
}

# 检查VS Code
Write-Host "`n[5] 检查VS Code..." -ForegroundColor Yellow
$vscodeVersion = code --version 2>$null
if ($vscodeVersion) {
    Write-Host "  ✅ $vscodeVersion" -ForegroundColor Green
} else {
    Write-Host "  ⚠️ VS Code 未安装（可选）" -ForegroundColor Yellow
}

Write-Host "`n=== 检查完成 ===" -ForegroundColor Cyan
```
