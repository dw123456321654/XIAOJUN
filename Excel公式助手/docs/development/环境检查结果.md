## 下一步操作

### 1. 安装Rust（必须）

Rust是Tauri后端的核心语言，必须安装才能开发。

**安装步骤：**

1. 访问 https://rustup.rs/
2. 下载 `rustup-init.exe`
3. 运行 `rustup-init.exe`
4. 按照提示安装（选择默认选项）
5. 重启终端
6. 验证安装：
   ```powershell
   rustc --version
   ```

**预期输出：**
```
rustc 1.70.0 (or higher)
```

### 2. 安装WebView2 Runtime（必须）

WebView2 Runtime是Tauri的依赖，必须安装。

**安装步骤：**

1. 访问 https://developer.microsoft.com/microsoft-edge/webview2/
2. 下载并安装 WebView2 Runtime
3. 验证安装：
   ```powershell
   reg query "HKLM\SOFTWARE\WOW6432Node\Microsoft\EdgeUpdate\Clients\{F3017226-FE2A-4295-8BDF-00C3A9A7E4C5}"
   ```

**预期输出：**
```
如果已安装，会显示版本信息
```

### 3. 安装VS Code插件（推荐）

打开VS Code，搜索并安装以下插件：

1. **Volar**
2. **Rust Analyzer**
3. **ESLint**
4. **Prettier**
5. **TailwindCSS IntelliSense**
